# 🌟 Nova Remains

Um jogo web baseado em Grand Chase, desenvolvido com HTML5, CSS3 e JavaScript.

## 🎮 Sobre o Jogo

Nova Remains é um jogo de aventura onde você pode:
- Escolher entre diferentes heróis
- Explorar mapas únicos
- Gerenciar inventário
- Realizar missões
- Comprar itens na loja

## 🚀 Como Jogar

1. Abra o arquivo `index.html` em seu navegador
2. Escolha seu herói favorito
3. Explore o mapa mundi
4. Complete missões e colete recompensas

## 🎨 Características Visuais

- **Design 3D** inspirado em pixel art
- **Interface holográfica** com efeitos visuais
- **Sistema de cores** consistente baseado nos heróis
- **Animações** suaves e responsivas

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com efeitos 3D e gradientes)
- JavaScript (ES6+)
- Sistema de sprites personalizado

## 📁 Estrutura do Projeto

```
GrandChase-Web/
```
